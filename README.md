# dailypilot

DailyPilot is a Flutter-based Note App designed to help users create, organize, and manage their notes easily.
The app also includes an **Online Public Notes session**, where users can share notes publicly and interact with shared content in a simple and accessible way.

## Features

* Create and manage personal notes
* Edit and delete saved notes
* Clean and simple user interface
* Offline note access for personal notes
* Online Public Notes session
* Public note sharing
* Built with Flutter for Android and cross-platform support

## About the Project

DailyPilot was created as a Flutter project to explore mobile app development, local note management, and online note-sharing features.

The main goal of the app is to provide a useful everyday tool where users can keep private notes while also having access to a shared online space for public notes.

## Tech Stack

* **Flutter**
* **Dart**
* Online database / backend service for public notes
* Android build support

## Installation

To run this project locally, make sure Flutter is installed on your machine.

### 1. Clone the repository

```bash
git clone https://github.com/your-username/dailypilot.git
```

### 2. Open the project folder

```bash
cd dailypilot
```

### 3. Install dependencies

```bash
flutter pub get
```

### 4. Run the app

```bash
flutter run
```

## Build APK

To generate an APK file:

```bash
flutter build apk
```

The APK will be available in:

```bash
build/app/outputs/flutter-apk/app-release.apk
```
